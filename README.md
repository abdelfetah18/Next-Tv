Next-tv is a web app for watching movies and series (tv-shows). it built in using the following web technologies:
 - NextJs.
 - TailWindCss.
 - SanityIO (Database).

--- 

## Getting Started

First, install dependencies:

```bash
npm install
```

then run the devlopement server:
```bash
npm run dev
```
